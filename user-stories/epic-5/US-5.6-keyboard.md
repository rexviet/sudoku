# US-5.6 — Keyboard Input

| Field | Value |
|---|---|
| **Epic** | 5 — UI Interactions & Game Flow |
| **Priority** | P1 (should-have) |
| **Story Points** | 3 |
| **Dependencies** | US-5.2 |

## User Story

**As a** player, **I want** dùng bàn phím để chơi **so that** tôi chơi nhanh hơn trên desktop.

## Tasks

- [ ] T1: Arrow keys (↑↓←→) để di chuyển selected cell
- [ ] T2: Phím 1-9 → nhập số (hoặc note nếu đang notes mode)
- [ ] T3: Backspace / Delete → erase
- [ ] T4: Ctrl+Z → undo
- [ ] T5: Phím N → toggle notes mode

## Acceptance Criteria

- [ ] AC1: Arrow keys di chuyển đúng hướng, không vượt biên grid
- [ ] AC2: Phím số hoạt động giống click number pad
- [ ] AC3: Shortcuts không conflict với browser shortcuts
- [ ] AC4: Keyboard input bị disable khi game paused
